# synstagram-scene-cocoapods-specs

Private CocoaPods spec repository for Synstagram scene pods.

## Usage

Add this spec repo as a source in your `Podfile`:

```ruby
source 'https://github.com/binaryloader/synstagram-scene-cocoapods-specs.git'
```
