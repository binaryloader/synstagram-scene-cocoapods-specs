# synstagram-scene-cocoapods-specs

Synstagram 화면 Pod를 위한 비공개 CocoaPods spec 저장소이다.

## Specs

| Pod | 최신 버전 | 설명 |
|-----|---------------|-------------|
| LoginScene | 1.0.10 | VIP 아키텍처 기반 로그인 화면 |

## 사용법

`Podfile`에 이 spec 저장소를 source로 추가한다.

```ruby
source 'https://github.com/binaryloader/synstagram-scene-cocoapods-specs.git'
```

## 관련 저장소

- https://github.com/binaryloader/synstagram-app
- https://github.com/binaryloader/synstagram-scene-login
