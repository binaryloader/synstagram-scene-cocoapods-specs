# synstagram-scene-cocoapods-specs

Synstagramのシーン用Podをホストするプライベートな CocoaPods spec リポジトリです。

## Specs

| Pod | 最新バージョン | 説明 |
|-----|---------------|-------------|
| LoginScene | 1.0.10 | VIPアーキテクチャによるログイン画面 |

## 使い方

`Podfile`にこの spec リポジトリを source として追加します。

```ruby
source 'https://github.com/binaryloader/synstagram-scene-cocoapods-specs.git'
```
